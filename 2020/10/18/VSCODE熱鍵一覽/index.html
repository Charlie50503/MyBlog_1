VS CODE熱鍵一覽

前言

這篇主要紀錄自我學習的部分，我會以我理解的方式去記述內容。

這次參加六角學院的網頁切版，基本功也要練好

藉由這次機會把平常不會刻意去訓練的部分也加強一下。

開始

vs code 是一個開發工具，主要用於寫 javascript , css ,html

利用各種開源套件，提高開發的效率

這次要來介紹的是 vs code 熱鍵的部分

類型

一般

ctrl+shirt+p可以下拉看到全部的指令

ctrl+p可透過搜尋列來跳轉到想要去的文件，需要自己打文件名

Ctrl+Shift+N可以另開一個新的 vs code 視窗

Ctrl+Shift+W可以關閉整個 VS CODE

無圖片

Ctrl+,要在點擊 vs code 視窗，的情況下才有用，在文件內是沒有效的，主要是打開「設定」頁面

Ctrl+K Ctrl+S可以開啟鍵盤快速鍵??

基礎

Alt+ ↑ / ↓把當前選擇的行做上下移動

Shift+Alt + ↓ / ↑將當前行往上或往下複製，這個很方便我常常用

也可以選擇一個區塊複製

Ctrl+Shift+K刪除當下行，不影響現在剪貼版保存的複製

Ctrl+Enter向下插入一行，不管輸入位置是否在行文字中間，一樣可以新增一行

Ctrl+Shift+Enter向上插入一行，不管輸入位置是否在行文字中間，一樣可以新增一行

Ctrl+Shift+\移至方括號，沒有測試成功

Ctrl+] / [移至方括號，沒有測試成功

Ctrl+Home移置文件頂部，也就是文件起點

Ctrl+End移置文件底部

Ctrl+↑ / ↓讓滾動條向上或向下移動

Alt+PgUp / PgDn切換當前頁面，跟 excel 一樣

Ctrl+Shift+[ / ]需要代碼長度夠長才有用，隱藏/顯示代碼區塊

Ctrl+K Ctrl+[ / ]隱藏顯示子區域??

Ctrl+K Ctrl+0全部展開??

Ctrl+K Ctrl+J移除行註解

Ctrl+K Ctrl+C加入行註解

Ctrl+/使用註解功能或是不使用

Shift+Alt+A使用區塊註解(可以切換)，好用!

Alt+Z切換是否自動換行

Ctrl+T可以搜尋內文或是文件名

Ctrl+G移置指定行，Debugger 好用!

Ctrl+Shift+O搜尋標籤名或是函數名稱

Ctrl+Shift+M切換至出問題的地方，需要主控台有跳出東西才可以

F8移動到當前文件出現警告的地方，好用!

Shift+F8移動到當前文件出現警告的地方(上一個)，好用!

Ctrl+Shift+Tab切換到上一次使用的文件

Alt+ ← / →跟Alt+PgUp / PgDn一樣
